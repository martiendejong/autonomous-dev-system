### Methode 1: ChatGPT Data Export (Meest Complete)

1. Ga naar https://chatgpt.com/
2. Klik op je profiel (rechtsboven)
3. Klik "Settings"
4. Ga naar "Data controls"
5. Klik "Export data"
6. Wacht op email met download link
7. Download en unzip
8. Upload `conversations.json` naar deze map

### Methode 2: Individuele Conversations (Snel)

1. Open een gesprek in ChatGPT
2. Klik op de titel van het gesprek
3. Kopieer de volledige tekst
4. Plak in een .txt bestand
5. Sla op met beschrijvende naam:
   ```
   2022-XX-XX_arjan_stroeve_chatbot_project.txt
   2023-XX-XX_social_media_hulp_discussion.txt
   2024-XX-XX_brand2boost_development.txt
   ```

### Methode 3: Screenshots

Als je geen tekst kunt exporteren:
1. Maak screenshots van belangrijke gesprekken
2. Sla op als PNG/JPG
3. Upload naar deze map
4. Claude kan images lezen en analyseren
